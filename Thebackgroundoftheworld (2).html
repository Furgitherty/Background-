How do you create a paragraph?

Paragraphs:
To create a paragraph please download Visual Studio Code (for computers) or DroidEdit (for phones). Then you should make a html file, by pressing "File" then pressing
"New File" then clicking "Text File" and then right clicking on the name of the file
and then going to your files and renaming the file you just made to "index.html". 		Now, type in <!DOCTYPE html>.
Then type in <html>, inside <html> type lang="en".
After that, type in </html>, which ends the <html> tag (a tag is like a keyword that define how webpages display their content there are three main parts of a tag, an opening tag, content, and a closing tag, the opening tag usually starts with a "<" and ends with a ">", the closing tag also starts with a "<" and then a "/", finally the closing tag ends with a ">", the content is whats inbetween the opening and closing tags).
Then make three spaces between <html> and </html> so you can type between them, you should also make spaces between every
tag you type so you can type in between the tags.
Then type inside of <body> , and then end the <body> tag with typing </body>.
Inside the body tag please type <title> and then end with </title>,
Inside the title tag please type what you want your website to be named, or leave it blank 		Finally, go to the end of the title tag, and make three spaces after it
Then please type in <p>, and then end the tag with </p>. To explain, the "p" tag, stands for paragraph, and in between the "p" tag you should write your paragraph 		This is the end of this tutorial, bless, and my hopes are for you to enjoy your day.
