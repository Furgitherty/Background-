How do you create a paragraph?

To create a paragraph please download Visual Studio Code (for computers) or DroidEdit (for phones).

Then you should make a html file, by pressing "File" then pressing
"New File" then clicking "Text File" and then right clicking on the name of the file
and then going to your files and renaming the file you just made to "index.html".

Now, type in <!DOCTYPE html>.
Then type in <html>, inside <html> type lang="en".
After that, type in </html>, which ends the <html> tag (a tag is like a keyword that define how webpages display their content there are three main parts of a tag, an opening tag, content, and a closing tag, the opening tag usually starts with a "<" and ends with a ">", the closing tag also starts with a "<" and then a "/", finally the closing tag ends with a ">", the content is whats inbetween the opening and closing tags).
Then make three spaces between <html> and </html> so you can type between them, you should also make spaces between every
tag you type so you can type in between the tags.
Then type inside of <body> , and then end the <body> tag with typing </body>.
Inside the body tag please type <title> and then end with </title>,
Inside the title tag please type what you want your website to be named, or leave it blank

Finally, go to the end of the title tag, and make three spaces after it
Then please type in <p>, and then end the tag with </p>.

Also always remember to put ";" after you finish typing in content.

You have now created your own paragraph tag, this is now the end of this tutorial
thank you for reading this.

The "p" tag in html: The "class" tag in html: The "div" tag in html: The "span" tag in html:
The "p" tag, stands for paragraph, and in between the "p" tag you can write your paragraph.
The "p" tag defines a block so whatever is in the "p" element is seperate from others. 		The "class" attribute modifies text in a tag, it is used to alter multiple tags by distributing content inside where it is defined.
The "class" attribute can be defined in the "style" tag, which is recommended to be added after the "title" tag, in between the "head"'s opening and closing tags
When you add the "style" tag, in between it type "." then a name, after that type "{}"(these are brackets where you put your content in within style),
and in between the "{}" you can type different attributes, one is "color" which can be defined by typing,
"color" then after that typing the color you would like to choose, then ending the line with ";". You can type "class=""" in an opening part of most tags,
Then in between the ""(quotation marks) after "class=" you can type in the name of your class to use your class. 		The "div" tag, stands for division, and it does just that, it divides the content inside of it from the content around it.
the "div" tag makes a block, which is how it divides what is inside of it from other content, though by itself does nothing, it's different from the "p" tag
because the "p" tag represents a paragraph while div represents a division.
It is more appropiate and fellow-coder-friendly to use a "div" tag than a "p" tag when seperating an element from others.
You can add classes to the "div" tag to customize what is inside the "div" tag 		The "span" tag is to modify a part of text in a tag.
Its very similar to the "div" tag because you can add classes to the "span" tag, though by itself the "span" tag does nothing
however, it does not divide the content inside of it from the content around it.
One of the uses of the "span" tag is to change text because it does not divide the content from the content around it.
